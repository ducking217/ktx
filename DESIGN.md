# Design System: Minimalist Tech Sophistication

## 1. Minimalism Principles
- **Whitespace (Negative Space):** Use generous padding and margins (Scale: 4px, 8px, 16px, 24px, 32px, 64px). Let the elements breathe.
- **Essentialism:** Remove every line, border, or decoration that doesn't serve a functional purpose.
- **Clutter-free:** Use "Progressive Disclosure" (chỉ hiện thông tin khi cần thiết).

## 2. Modern SaaS Aesthetic
- **Depth & Elevation:** Instead of 1px borders, use multi-layered soft shadows.
  - *Shadow Example:* `0 1px 2px rgba(0,0,0,0.05), 0 4px 6px rgba(0,0,0,0.02)`.
- **Palette:** Use "Tinted Neutrals" (Slate or Zinc). 
  - Background: `#F8FAFC`.
  - Cards: `#FFFFFF`.
  - Primary Accent: Indigo-600 or Blue-600 (used sparingly).
- **Typography:** Use Geometric Sans-serif (Inter, Be Vietnam Pro).
  - Use `tabular-nums` for data to ensure alignment.
- **Micro-borders:** Use `border-slate-200/50` for very subtle separation.

## 3. The Hybrid Rule
Combine the "void" of minimalism with the "precision" of SaaS tools. Every component should feel like a high-end software tool (Pro Tool).
